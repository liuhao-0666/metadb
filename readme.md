METADB 是一个面向数据中心模型及测点的管理平台。



METADB提供了自定义模型管理，用户可以实现自定义的空间关系，同样可以根据企业情况来建立出数据中心各类设备模型及监控测点。其中模型的属性可以自由增加，这就意味着用户可以自定义模型的各类属性，来实现灵活多变的需求，创建完模型后，可以根据模型创建出实例。企业的模型管理，设备管理，都可以放入METADB中。
